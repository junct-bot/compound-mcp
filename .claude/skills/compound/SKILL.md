# Compound MCP Server

Hosted MCP server for Compound, powered by Junct.

## Connect

```json
{
  "mcpServers": {
    "compound": {
      "url": "https://compound.mcp.junct.dev/mcp",
      "transport": "streamable-http"
    }
  }
}
```
